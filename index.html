Maven实战读书笔记

读书笔记

坐标和依赖

依赖的范围

  • compile 编译依赖范围。默认范围，对于编译、测试、运行三种classpath都有效
  • test 测试依赖范围。只对测试classpath有效，如：JUnit
  • provided 已提供依赖范围。对于编译和测试classpath有效，如：servlet-api
  • runtime 运行时依赖范围。对于测试和运行classpath有效。如：JDBC驱动
  • system 系统依赖范围
  • import 导入依赖范围
