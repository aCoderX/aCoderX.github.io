spring, java

spring-boot简介

spring-boot旨在帮助开发者更容易地创建基于Spring的应用程序和服务，使得现有的和新的Spring开发者能够最快速地获得所需要的Spring功能。

Spring Boot不生成代码，且完全不需要XML配置。
